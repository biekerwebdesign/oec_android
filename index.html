Open Exoplanet Catalgue Android App

An Android app that displays exoplanet data from the Open Exoplanet Catalogue

Download this project as a .zip file Download this project as a tar.gz file

Based off the Open Exoplanet Catalogue, this app is designed to act as a portal for the Catalogue's data. It's ultimate goal is to be at least on par with the iPhone application for the Open Exoplanet Catalogue. Feel free to contribute! The app will eventually be published on the Google Play store, Amazon Appstore, and F-Droid.
